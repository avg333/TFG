# TFG
TFG - GITT 2019-2020: Simulador red 5G eficiente.

Trabajo Fin de Grado del Grado en Ingeniería de las Tecnologías de las Telecomunicaciónes
Especialidad de Telemática. Curso 2019-2020.

El programa consiste en un simulador de una red 5G donde las estaciones base usan ASM con el fin de mejorar la eficiencia del sistema.

La compilación se realiza con la siguiente instrucción:
ant compile

Para mostrar la información de ejecución de cualquier entidad se debe ejecutar con el parametro:
-h|--help

Para ejecutar una simulación se debe lanzar el Broker y después todas las entidades UE y BS que participen en la simulación, configurando todas las entidades por línea de parámetros. Una vez lanzadas, se inicia la simulación con la tecla "Enter".
